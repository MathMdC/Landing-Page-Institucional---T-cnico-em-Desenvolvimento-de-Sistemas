# Landing Page Institucional - Técnico em Desenvolvimento de Sistemas
Uma página responsiva criada para apresentar o curso e seus aspectos.

### 🛠️ Linguagens Utilizadas:
- HTML5
- JavaScript
- CSS

### ⚙️ Funcionalidades:
- Menu lateral responsivo.
- Carrosel de slides.
- Formulário de Contato.

### 📚 O que aprendi:
- Comunicação de Dados.
- Navegação Single-Page.
- Criar uma página responsiva.
- Integração de bibliotecas externas.

### 🚀 Como rodar o projeto:
1. Instale os arquivos e deixe a pasta `img` junto ao arquivo `index.html`.
2. Abra o `index.html` e pronto!
